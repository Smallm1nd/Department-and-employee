# API Организационной структуры

REST API сервис для управления иерархией департаментов и сотрудниками. Выполнено в рамках тестового задания.

## Стек технологий
* **Язык:** Go (Golang) 1.22+
* **Роутинг:** Встроенный `net/http` (ServeMux)
* **База данных:** PostgreSQL
* **ORM:** GORM
* **Миграции:** Goose
* **Инфраструктура:** Docker, docker-compose

## Запуск проекта

Для запуска приложения требуется только установленный Docker Desktop (или Docker Engine).

1. Перейдите в корневую директорию проекта.
2. Выполните команду для сборки и запуска контейнеров:
   ```bash
   docker-compose up --build
